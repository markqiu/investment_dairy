| 101  | 财务       | 是否有持续的正现金流？                                       | 利润容易造假调节，企业是一个从现金开始到现金结束的游戏。**净现比，意思是净利润现金含量，即经营现金流量净额/净利润，就是说企业实现一元净利润实际流入多少现金。**从原理上来看，经营活动现金流出除了支付购进的原材料、人工成本、税费之外，并不包含固定资产折旧、利息支出等内容，即经营活动现金流出小于营业成本+期间费用+所得税（利润表中的成本项）。如果经营活动现金流流入等于营业收入，而经营活动现金流出小于成本项，那么必然得出结论：**净利润应小于经营活动现金流量净额，即净现比应该大于1**。长周期来看（10年以上），经营活动现金流量净额/净利润应该大于1，当然越大越好。看整个A股的净现比，扣除金融业之后，A股实现净利润1.94万亿元，经济活动现金流量净额为2.55万亿元，净现比为1.32，也就意味着A股实现1元的净利润，那么流入的现金为1.32元。<br />作者：初善君 链接：https://xueqiu.com/6615553088/138190953 |          |      |

| 305  | 公司       | 公司是否具有反脆弱性?                                        | “反脆弱”一词来源于塔勒布的《反脆弱》一书，书中塔勒布提到的关于反脆弱的概念，认为很多大型企业、组织甚至金融系统是非常脆弱的，在负面的黑天鹅事件下不堪一击；相反，另一些事物却能从冲击中受益。它们经常暴露在各种波动性、随机性、混乱和压力下，反而茁壮成长。“反脆弱性”（antifragile）一词正用来描述这类事物。战略规划是一种迷信，生存其实取决于适应性和环境条件的相互作用。<br />书中举例兄弟两个，老大安稳的白领，生活按部就班。老二出租车司机，靠跑车挣钱。但一年算下来，兄弟两个挣的钱差不多。但老二的工作相对于老大就具有反脆弱性，可能某一天他一分钱挣不到，但他在随后的一些日子里却能够将其补回来。对于老二来说每时每刻都面临着小的不确定性，这些小的不确定性对他的整体影响不大，却造就了他的工作的相对稳定性。而老大的体面工作，完全依赖于公司，当被公司开除或公司倒闭，他便失去了工作，失去了身份。他什么时候失去工作就是一个大的不确定性。因此，他的工作不具有反脆弱性。 |          |      |
